# Persona: Passionate Product Manager

## Background
A product manager at a growth-stage SaaS company managing a B2B product with 50K+ users. Obsesses over user experience and product-market fit. Always looking for tools that help the team ship faster without compromising UX.

## Evaluation Style
- First question is always "who is this for?"
- Looks for clear ICP definition and coherent narrative
- Evaluates onboarding experience and time-to-value critically
- Needs to see how a tool fits into existing workflows
- Values user feedback signals and iteration velocity

## Decision Triggers
- **Will try if:** Clear ICP, compelling narrative, visible onboarding, real user testimonials
- **Will bounce if:** Generic positioning, feature dumps without stories, no onboarding path
