# Product Hunt Grinder Persona Ticket Pack

This folder contains **100 tickets** for generating personas who:
- Browse Product Hunt very frequently and invest lots of time
- Are highly performative/aspirational in tone
- Tend to be **low-rigor** in evaluation (buzzword- and social-proof-driven)

Rules:
- Write the final persona doc in English using `PERSONA.md` sections 0-10 (fully filled).
- Keep it concrete: numbers, named tools, explicit constraints, and realistic behaviors.
- Persona voice block must be 2nd person and 80-160 words.

Output location:
- `personas/<persona_id>.md`

Naming:
- Persona IDs in this pack use the `ph_grinder_` prefix to avoid collisions.
