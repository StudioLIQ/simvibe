# Elite Persona Ticket Pack

This folder contains **100 tickets** for generating *high-agency, ambitious, top-tier* personas.

Rules:
- Write the final persona doc in English using `PERSONA.md` sections 0-10 (fully filled).
- Keep it concrete: numbers, named tools, explicit constraints, and realistic behaviors.
- Persona voice block must be 2nd person and 80-160 words.

Output location:
- `personas/<persona_id>.md`

Naming:
- Persona IDs in this pack use the `elite_` prefix to avoid collisions.
